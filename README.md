## Realtime-Character-Counter
A simple web application that allows users to type a message in a text area and displays a real-time character counter. The counter updates automatically as the user types and shows how many characters have been used and how many are remaining.

## Features
- Real-time character counting
- Shows typed characters and remaining characters
- Maximum character limit (200 characters)
- Prevents typing after reaching the limit
- Displays a warning message when the limit is reached

## Technologies Used
- **HTML** – Structure of the webpage  
- **CSS** – Styling and layout design  
- **JavaScript** – Real-time character counting logic 

## How It Works
1. The user types a message in the text area.
2. JavaScript detects the input event in real time.
3. The character counter updates automatically.
4. When the character limit is reached, further typing is prevented and a warning message appears.

## Purpose of the Project
This project is created to practice **JavaScript event handling and DOM manipulation**. It is also useful for beginners who want to understand how real-time input tracking works in web applications.
